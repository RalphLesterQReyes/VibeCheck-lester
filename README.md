# VibeCheck-lester
install node.js

npm init -y
npm install express cors

then run npm index.js

Endpoint list (open it on your browser): 
1. http://localhost:3000/api/fortune (JSON).
2. http://localhost:3000/api/joke (JSON).
3. http://localhost:3000/api/vibe?mood=happy (JSON).
4. http://localhost:3000/api/smashes (JSON).

RUN THE CODE
- run the node under backend folder = node index.js
- run the html under frontend folder = indext.html
  
- note: when you run your frontend you need to run the node on your terminal first.
